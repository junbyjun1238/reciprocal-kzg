# reciprocal-kzg

Repository layout for the reciprocal KZG backend package.

- `poc/`: implementation and integration code
- `benchmark/`: comparison runners and benchmark outputs
- `docs/`: core paper PDF, milestones, and reviewer-facing notes

## Reading Order

1. `docs/reciprocal_poc.md`
2. `docs/m4_milestones.md`
3. `poc/crates/ivc/examples/reciprocal_poc.rs`
4. `benchmark/reciprocal_snapshot.csv`
